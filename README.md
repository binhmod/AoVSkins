# AoVSkins
Ứng dụng hỗ trợ cài đặt mod skin liên quân thông qua Shizuku mà không cần root.
# Lưu ý
• App cũng chỉ copy file mod vô data game như các app thông thường khác.

• Sử dụng APK chính thức từ mình để tránh bị cài mã độc/khóa acc v.v

• Cài mod phải uy tín để tránh bị khóa acc mà đổi thừa là do app :v

• Mình không chịu trách nhiệm về việc bị khóa tài khoản game :<

• Ứng dụng mã nguồn mở, sạch sẽ các bạn có thể tham khảo nên yên tâm nhé :3
# Chức năng
• Có thể cài mod gộp (cài nhiều mod cùng lúc)

• Nhanh, đơn giản, gọn lẹ và an toàn

• Backup tài nguyên - Xóa tài nguyên

• Thế thôi :3
